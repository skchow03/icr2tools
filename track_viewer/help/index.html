Overview

Track Viewer is a desktop tool for inspecting and editing IndyCar Racing II (ICR2) track data. It is designed for track builders and modders who want precise, visual control over cameras, AI lines, and related metadata.

What Track Viewer does

  • Displays TRK geometry, centerlines, and surface bounds
  • Visualizes AI driving lines (LP files)
  • Allows interactive and numeric camera editing
  • Parses and edits selected track.txt parameters
  • Safely saves changes with automatic backups

What Track Viewer does not do

  • It does not simulate physics or AI behavior
  • It does not modify files unless you explicitly save
  • It does not replace in-game testing
This manual is bundled with the application and matches the exact version you are running. No internet connection is required.

Getting Started

To begin, select a valid ICR2 track folder. Track Viewer will automatically detect available files and load what it can safely interpret.

Minimum required files

  • .TRK file
  • Centerline data (if present)

Optional but supported

  • .LP AI line files
  • .CAM / .SCR camera files
  • track.txt (PIT and metadata)
  • DAT-embedded camera data (fallback)

If a file is missing or unreadable, the viewer will continue loading with reduced functionality. Errors are reported in the status area and logs.

Start by familiarizing yourself with the Preview Window. Most workflows begin there.
